## HTML A-Z চিটশিট (বাংলা)

### ১. <html>
**ব্যবহার**: HTML ডকুমেন্টের মূল ট্যাগ।
**উদাহরণ**:
```html
<html>
  <head></head>
  <body></body>
</html>
```

### ২. <head>
**ব্যবহার**: মেটাডেটা বা শিরোনাম সংক্রান্ত তথ্য রাখে।
**উদাহরণ**:
```html
<head>
  <title>আমার পৃষ্ঠা</title>
</head>
```

### ৩. <title>
**ব্যবহার**: ব্রাউজারের ট্যাবে শিরোনাম প্রদর্শন করে।
**উদাহরণ**:
```html
<title>আমার ওয়েবসাইট</title>
```

### ৪. <body>
**ব্যবহার**: ওয়েবপেজের প্রধান কন্টেন্ট রাখে।
**উদাহরণ**:
```html
<body>
  <h1>স্বাগতম</h1>
  <p>এটি একটি প্যারাগ্রাফ।</p>
</body>
```

### ৫. <h1> থেকে <h6>
**ব্যবহার**: হেডিং টেক্সটের জন্য ব্যবহৃত হয়। <h1> সবচেয়ে বড়, <h6> সবচেয়ে ছোট।
**উদাহরণ**:
```html
<h1>বড় হেডিং</h1>
<h6>ছোট হেডিং</h6>
```

### ৬. <p>
**ব্যবহার**: প্যারাগ্রাফ তৈরি করতে।
**উদাহরণ**:
```html
<p>এটি একটি সাধারণ প্যারাগ্রাফ।</p>
```

### ৭. <a>
**ব্যবহার**: হাইপারলিঙ্ক তৈরি করতে।
**উদাহরণ**:
```html
<a href="https://www.google.com">গুগলে যান</a>
```

### ৮. <img>
**ব্যবহার**: ইমেজ প্রদর্শন করতে।
**উদাহরণ**:
```html
<img src="image.jpg" alt="একটি ছবি">
```

### ৯. <ul>, <ol>, এবং <li>
**ব্যবহার**: তালিকা তৈরি করতে।
**উদাহরণ**:
```html
<ul>
  <li>আইটেম ১</li>
  <li>আইটেম ২</li>
</ul>

<ol>
  <li>প্রথম</li>
  <li>দ্বিতীয়</li>
</ol>
```

### ১০. <table>, <tr>, <td>, এবং <th>
**ব্যবহার**: টেবিল তৈরি করতে।
**উদাহরণ**:
```html
<table border="1">
  <tr>
    <th>নাম</th>
    <th>বয়স</th>
  </tr>
  <tr>
    <td>রহিম</td>
    <td>২৫</td>
  </tr>
</table>
```

### ১১. <div>
**ব্যবহার**: ব্লক লেভেলের কন্টেইনার।
**উদাহরণ**:
```html
<div>
  <h2>এইচটিএমএল</h2>
  <p>এটি একটি ব্লক।</p>
</div>
```

### ১২. <span>
**ব্যবহার**: ইনলাইন লেভেলের কন্টেইনার।
**উদাহরণ**:
```html
<p>এই লেখা <span style="color: red;">লাল</span> রঙের।</p>
```

### ১৩. <form>
**ব্যবহার**: ফর্ম তৈরি করতে।
**উদাহরণ**:
```html
<form action="submit.php" method="post">
  <label for="name">নাম:</label>
  <input type="text" id="name" name="name">
  <button type="submit">জমা দিন</button>
</form>
```

### ১৪. <br>
**ব্যবহার**: লাইনে ব্রেক দিতে।
**উদাহরণ**:
```html
<p>এই লাইনটি<br>দুই ভাগে বিভক্ত।</p>
```

### ১৫. <hr>
**ব্যবহার**: একটি অনুভূমিক রেখা যোগ করতে।
**উদাহরণ**:
```html
<p>উপরের অংশ</p>
<hr>
<p>নিচের অংশ</p>
```

### ১৬. <strong> এবং <em>
**ব্যবহার**: <strong> বোল্ড এবং <em> ইটালিক টেক্সটের জন্য।
**উদাহরণ**:
```html
<p><strong>গুরুত্বপূর্ণ</strong> লেখা।</p>
<p><em>ইটালিক</em> লেখা।</p>
```

### ১৭. <audio>
**ব্যবহার**: অডিও যোগ করতে।
**উদাহরণ**:
```html
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  আপনার ব্রাউজার অডিও সমর্থন করে না।
</audio>
```

### ১৮. <video>
**ব্যবহার**: ভিডিও যোগ করতে।
**উদাহরণ**:
```html
<video controls width="500">
  <source src="video.mp4" type="video/mp4">
  আপনার ব্রাউজার ভিডিও সমর্থন করে না।
</video>
```

### ১৯. <iframe>
**ব্যবহার**: অন্য পেজ এমবেড করতে।
**উদাহরণ**:
```html
<iframe src="https://www.example.com" width="600" height="400"></iframe>
```

### ২০. <style>
**ব্যবহার**: ইন্টারনাল CSS যোগ করতে।
**উদাহরণ**:
```html
<style>
  body {
    background-color: lightblue;
  }
</style>
```

এই চিটশিটে HTML-এর প্রাথমিক এবং গুরুত্বপূর্ণ ট্যাগগুলোর সাথে উদাহরণ দেয়া হয়েছে। এটি ব্যবহার করে দ্রুত HTML শিখতে পারবেন।
